# Practice Exercises

This directory contains practice problems and exercises for skill development.

## Exercise Categories

### Drill Practice
- Repetitive practice for skill building
- Graduated difficulty
- Immediate feedback

### Application Problems
- Real-world scenarios
- Multi-step solutions
- Critical thinking

### Challenge Problems
- Extension and enrichment
- Advanced applications
- Creative problem solving

## Exercise Format
```json
{
  "exercise_id": "alg_linear_001",
  "topic": "linear_equations",
  "difficulty": "easy",
  "type": "drill",
  "problem": "Solve for x: x + 7 = 12",
  "solution": "x = 5",
  "steps": [
    "Subtract 7 from both sides",
    "x + 7 - 7 = 12 - 7",
    "x = 5"
  ],
  "hints": [
    "What operation undoes addition?",
    "Apply the same operation to both sides",
    "Simplify both sides"
  ],
  "variations": [
    "x + 3 = 10",
    "x + 15 = 22"
  ],
  "estimated_time": 2
}
```

## Difficulty Progression

- **Beginner**: Basic concepts, simple problems
- **Easy**: Standard problems with clear methods
- **Medium**: Multi-step problems, some complexity
- **Hard**: Complex problems, multiple approaches
- **Advanced**: Challenging extensions, proofs

## Creating Exercises

1. Start with learning objective
2. Create graduated difficulty levels
3. Include worked examples
4. Provide multiple hints
5. Add solution explanations
6. Create variations for practice
